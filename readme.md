## about ProcessFilter
- 忽略表和指定表都配置时：
    1. 当指定表和忽略表都符合条件则指定表优先，进行保留
    2. 当仅指定表符合条件时，则保留，其余表丢弃。
    3. 当仅忽略表符合条件时，则丢弃，其余表保留。
- 仅忽略表配置时：符合条件的丢弃，其余保留
- 仅指定表配置时：符合条件的保留，其余丢弃
